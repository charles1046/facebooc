What is facebooc

Facebooc is a social platform implemented in C language. Based on C, its response time is much less than others', which performs the flexibility of C.

The website is designed with Boostrap components and SCSS stylesheets. The latters are both edited and recognized more easily than CSS ones are.

We use SQLite3 as database. SQLite supports most OSes, such as Linux, MacOS, and Windows. It also provides easy-to-use APIs.

About Boostrap About SQLite3

Prerequisites

  1. Install SQLite development package:
    sudo apt-get install libsqlite3-dev
  2. Build and Run:
    cd facebooc
    make run
